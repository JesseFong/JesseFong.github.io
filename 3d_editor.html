3D Editor

This 3D editor is a framework that focused on the ability to create and manipulate data within a 3D engine. So far the editor has the ability to hotload and parse data files (Shaders, Levels, Materials, etc...). It also is able to save and manipulate objects within levels (translate, scale, rotate) with an integrated immediate-mode ui that is both 2D and 3D (The manipulator is treated as a ui object that relies on the mouse raycast rather then the mouse position on the screen).
