Koi - A 2D Game Engine

Koi is a 2D game engine written from scratch and inspired by Casey Muratori's Handmade Hero series. It was written from scratch and depends on just the C runtime library and OS and Sean Barrett's stb Libraries

The goal of the project was to learn many of the requirements of building a modern 2D game engine.
